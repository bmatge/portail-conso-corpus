# Le professionnel n'a pas fourni les informations précontractuelles obligatoires : que faire ?

## En bref
Si un professionnel ne vous a pas fourni les informations obligatoires avant la conclusion d'un contrat, vous pouvez agir. Ces informations incluent les caractéristiques du bien ou service, le prix, les modalités de livraison, et les coordonnées du professionnel. Vous pouvez signaler ce manquement et, dans certains cas, annuler le contrat ou refuser de payer des frais supplémentaires non mentionnés.

## De quoi s'agit-il ?
Les informations précontractuelles sont des informations que le professionnel doit vous fournir avant la conclusion d'un contrat. Elles concernent les caractéristiques essentielles du bien ou service, le prix, les modalités de livraison, et les coordonnées du professionnel. Ces informations sont obligatoires pour tous les contrats conclus avec des consommateurs, que ce soit en ligne, hors établissement (comme à domicile) ou en magasin.

**Ce qui est couvert** :
- Les contrats conclus à distance (en ligne)
- Les contrats conclus hors établissement (à domicile, en foire, etc.)
- Les contrats conclus en magasin

**Ce qui n'est pas couvert** :
- Les contrats conclus dans les foires et salons où le droit de rétractation ne s'applique pas
- Les contrats conclus avec des professionnels (B2B)

## Quels sont vos droits ?
Le professionnel doit vous fournir toutes les informations nécessaires pour que vous puissiez prendre une décision éclairée avant de conclure un contrat. Si ces informations ne sont pas fournies, vous avez plusieurs recours. Par exemple, si les frais supplémentaires ne sont pas mentionnés, vous n'êtes pas tenu de les payer. De plus, si le professionnel ne respecte pas ses obligations d'information, vous pouvez demander l'annulation du contrat ou signaler le manquement aux autorités compétentes.

## Que faire concrètement ?
1. **Demandez les informations manquantes** : Contactez le professionnel pour obtenir les informations qui n'ont pas été fournies. Vous pouvez le faire par email, téléphone ou courrier.
2. **Conservez les preuves** : Gardez une trace de toutes les communications avec le professionnel, y compris les emails, les relevés téléphoniques et les copies des documents reçus.
3. **Signalez le manquement** : Si le professionnel ne fournit pas les informations demandées, signalez le manquement à la DGCCRF via la plateforme [SignalConso](https://signal.conso.gouv.fr).
4. **Recours ultime** : Si les étapes précédentes n'aboutissent pas, vous pouvez saisir un médiateur sectoriel ou engager une action en justice pour faire valoir vos droits.

## Exemples / cas concrets
**Situation** : Vous avez acheté un produit en ligne et le professionnel n'a pas mentionné les frais de livraison supplémentaires.
**Ce que vous pouvez faire** : Vous pouvez refuser de payer ces frais supplémentaires et signaler le manquement à la DGCCRF via [SignalConso](https://signal.conso.gouv.fr).
**Résultat attendu** : Le professionnel doit annuler les frais de livraison supplémentaires ou vous rembourser si vous avez déjà payé.

**Situation** : Vous avez conclu un contrat de service à domicile et le professionnel n'a pas fourni les conditions générales de vente.
**Ce que vous pouvez faire** : Demandez les conditions générales de vente par écrit. Si le professionnel ne les fournit pas, signalez le manquement à la DGCCRF via [SignalConso](https://signal.conso.gouv.fr).
**Résultat attendu** : Le professionnel doit fournir les conditions générales de vente ou risque des sanctions.

**Situation** : Vous avez acheté un produit dans une foire et le professionnel n'a pas mentionné le droit de rétractation.
**Ce que vous pouvez faire** : Informez-vous sur vos droits et, si nécessaire, signalez le manquement à la DGCCRF via [SignalConso](https://signal.conso.gouv.fr).
**Résultat attendu** : Le professionnel doit être informé de ses obligations et risque des sanctions si elles ne sont pas respectées.
