# Honoraires du syndic : comment contester des factures opaques ou abusives ?

## En bref
Votre syndic facture des prestations non prévues au contrat ou des montants excessifs ? Vous pouvez contester ces honoraires. Vérifiez votre contrat, demandez des justifications et, si nécessaire, saisissez le médiateur ou un juge.

## De quoi s'agit-il ?
Les honoraires du syndic doivent être clairement définis dans un contrat type obligatoire depuis la loi Alur (2014). Ce contrat précise les prestations incluses dans le forfait et celles facturées en supplément. Certains syndics facturent des prestations non prévues, majorent indûment des tarifs ou omettent d'inclure des services dans le forfait.

**Qui est concerné ?**
Tous les copropriétaires dont le syndic ne respecte pas le contrat type ou facture des prestations abusives.

**Ce qui est couvert**
- Prestations incluses dans le forfait mais facturées en supplément
- Majorations indues de tarifs
- Facturation de prestations particulières non justifiées
- Non-respect du contrat type

**Ce qui n'est pas couvert**
- Les honoraires conformes au contrat type
- Les prestations particulières clairement acceptées par l'assemblée générale

## Quels sont vos droits ?
Le contrat de syndic doit respecter un modèle type imposé par la loi. Les honoraires pour prestations particulières ne peuvent être facturés sans accord préalable de l'assemblée générale. Les majorations de tarifs doivent être justifiées.

**Obligations du syndic**
- Utiliser le contrat type obligatoire
- Inclure certaines prestations dans le forfait (ex : organisation des assemblées générales)
- Facturer les prestations particulières uniquement si elles ont été votées en assemblée générale
- Justifier toute majoration de tarif

**Délais importants**
- Le syndic doit fournir un état détaillé des sommes perçues annuellement
- Vous pouvez contester les honoraires dans un délai de 5 ans à partir de leur facturation

## Que faire concrètement ?
1. **Vérifiez votre contrat** : comparez les factures reçues avec les prestations prévues dans le contrat type. Notez les écarts.
2. **Demandez des justifications** : envoyez un courrier recommandé avec accusé de réception au syndic pour demander des explications sur les factures litigieuses.
3. **Saisissez le conseil syndical** : informez le conseil syndical des anomalies constatées et demandez son intervention.
4. **Engagez une médiation** : si le syndic ne répond pas ou maintient ses factures, saisissez le médiateur de la consommation.
5. **Saisissez un juge** : si la médiation échoue, vous pouvez engager une action en justice pour faire annuler les factures abusives.

## Exemples / cas concrets

**Situation** : "Mon syndic facture des frais de reprographie pour les documents de l'assemblée générale annuelle."
**Ce que vous pouvez faire** : Vérifiez votre contrat. Les frais de reprographie doivent être inclus dans le forfait. Demandez au syndic de justifier cette facturation ou de l'annuler.
**Résultat attendu** : Le syndic annule la facture ou vous fournissez une justification valable.

**Situation** : "Le syndic facture des heures supplémentaires pour les assemblées générales au-delà de la plage horaire prévue."
**Ce que vous pouvez faire** : Consultez votre contrat. Les heures supplémentaires doivent être déterminées selon un coût horaire non majoré. Contestez cette facturation auprès du syndic.
**Résultat attendu** : Le syndic rectifie la facture ou justifie la majoration.

**Situation** : "Le syndic facture des frais d'ouverture de compte bancaire séparé alors que cette prestation doit être incluse dans le forfait."
**Ce que vous pouvez faire** : Envoyez un courrier recommandé au syndic pour contester cette facturation. Mentionnez l'article du contrat type qui prévoit cette prestation dans le forfait.
**Résultat attendu** : Le syndic annule la facture ou vous fournissez une explication conforme à la loi.
