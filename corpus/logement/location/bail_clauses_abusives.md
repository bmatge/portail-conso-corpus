# Mon bail contient des clauses abusives : que faire ?

## En bref
Votre bail ne peut pas contenir de clauses abusives. Si c'est le cas, vous pouvez demander leur suppression. Le professionnel doit les retirer. Vous pouvez signaler ces clauses abusives sur [SignalConso](https://signal.conso.gouv.fr).

## De quoi s'agit-il ?
Un bail de location est un contrat entre vous (le locataire) et le propriétaire (le bailleur). Certaines clauses dans ce contrat peuvent être abusives. Une clause est abusive lorsqu'elle crée un déséquilibre significatif entre vos droits et obligations et ceux du bailleur.

Les clauses abusives sont interdites par la loi. Elles ne sont pas valables, même si vous les avez signées. Vous pouvez les faire supprimer du contrat.

Cette fiche concerne les clauses abusives dans les baux de location d'un logement. Elle ne concerne pas les locations saisonnières ou les locations de vacances.

## Quels sont vos droits ?
La loi interdit les clauses abusives dans les contrats de location. Si votre bail contient une clause abusive, vous pouvez demander sa suppression. Le professionnel doit la retirer.

Les clauses abusives sont listées dans des textes officiels. Par exemple, une clause qui interdit d'héberger un tiers ou qui impose une assurance spécifique peut être abusive.

Si le bailleur inclut une clause abusive dans le contrat, il commet une infraction. Vous pouvez porter plainte ou signaler la clause abusive sur [SignalConso](https://signal.conso.gouv.fr).

## Que faire concrètement ?
1. **Identifier la clause abusive** : Lisez attentivement votre bail. Repérez les clauses qui vous semblent abusives.
2. **Consulter un professionnel** : Contactez un conseiller en droit du logement ou une association de consommateurs pour vérifier si la clause est bien abusive.
3. **Demander la suppression de la clause** : Envoyez une lettre recommandée avec accusé de réception au bailleur pour lui demander de supprimer la clause abusive.
4. **Signalement** : Si le bailleur ne supprime pas la clause, signalez-la sur [SignalConso](https://signal.conso.gouv.fr).
5. **Recours ultime** : Si le signalement ne donne pas de résultat, vous pouvez saisir le tribunal judiciaire pour faire valoir vos droits.

## Exemples / cas concrets

**Situation** : Votre bail contient une clause interdisant d'héberger un tiers.
**Ce que vous pouvez faire** : Cette clause est abusive. Vous pouvez demander sa suppression au bailleur. Si le bailleur refuse, vous pouvez signaler la clause sur [SignalConso](https://signal.conso.gouv.fr).
**Résultat attendu** : Le bailleur doit supprimer la clause abusive. Vous pourrez héberger des tiers sans enfreindre le contrat.

**Situation** : Votre bail contient une clause imposant une assurance spécifique.
**Ce que vous pouvez faire** : Cette clause est abusive. Vous pouvez demander sa suppression au bailleur. Si le bailleur refuse, vous pouvez signaler la clause sur [SignalConso](https://signal.conso.gouv.fr).
**Résultat attendu** : Le bailleur doit supprimer la clause abusive. Vous pourrez choisir votre assurance sans être contraint par le contrat.

**Ce que vous pouvez faire** : Cette clause est abusive. Vous pouvez demander sa suppression au bailleur. Si le bailleur refuse, vous pouvez signaler la clause sur [SignalConso](https://signal.conso.gouv.fr).
**Résultat attendu** : Le bailleur doit supprimer la clause abusive. Vous serez protégé contre une résiliation automatique sans mise en demeure.
