# Mutuelle santé : remboursements inférieurs aux garanties annoncées

## En bref
Votre mutuelle ne rembourse pas comme prévu ? Vous avez des droits. Vérifiez votre contrat, contactez votre mutuelle et, si nécessaire, saisissez le médiateur. Les délais de carence et les exclusions de garantie peuvent expliquer des remboursements insuffisants.

## De quoi s'agit-il ?
Vous avez souscrit une mutuelle santé et les remboursements sont inférieurs à ceux prévus dans votre contrat. Cela peut concerner des soins médicaux, des frais d'hospitalisation, des prothèses dentaires, des lunettes ou d'autres prestations. Cette fiche vous explique vos droits et les démarches à suivre.

## Quels sont vos droits ?
Votre contrat de mutuelle doit préciser les garanties et les taux de remboursement. Les professionnels doivent respecter ces engagements. Les délais de carence (périodes pendant lesquelles certaines prestations ne sont pas remboursées) et les exclusions de garantie doivent être clairement indiqués dans votre contrat.

## Que faire concrètement ?
1. **Vérifiez votre contrat** : Consultez les conditions générales pour comprendre les garanties, les délais de carence et les exclusions.
2. **Contactez votre mutuelle** : Demandez des explications sur les remboursements insuffisants. Fournissez tous les justificatifs nécessaires.
3. **Envoyez une réclamation écrite** : Si la réponse de votre mutuelle ne vous satisfait pas, envoyez une lettre recommandée avec accusé de réception en détaillant votre problème.
4. **Saisissez le médiateur** : Si aucune solution n'est trouvée, contactez le médiateur de l'assurance via [le site de la médiation de l'assurance](https://www.mediation-assurance.org).

## Exemples / cas concrets
**Situation** : Vous avez souscrit une mutuelle qui promet un remboursement à 100 % pour les soins dentaires, mais vous devez payer 20 % de la facture.
**Ce que vous pouvez faire** : Vérifiez votre contrat pour confirmer les garanties. Contactez votre mutuelle pour obtenir des explications. Si nécessaire, saisissez le médiateur de l'assurance.
**Résultat attendu** : Votre mutuelle doit vous rembourser conformément aux garanties annoncées.

**Situation** : Votre mutuelle ne rembourse pas les frais d'hospitalisation pendant les 6 premiers mois de votre contrat.
**Ce que vous pouvez faire** : Vérifiez les délais de carence dans votre contrat. Si les délais sont respectés, contactez votre mutuelle pour obtenir un remboursement.
**Résultat attendu** : Votre mutuelle doit vous rembourser conformément aux garanties annoncées après la période de carence.

**Situation** : Votre mutuelle refuse de rembourser des soins d'optique alors que votre contrat prévoit une prise en charge.
**Ce que vous pouvez faire** : Vérifiez les exclusions de garantie dans votre contrat. Si les soins sont couverts, contactez votre mutuelle pour obtenir un remboursement.
**Résultat attendu** : Votre mutuelle doit vous rembourser conformément aux garanties annoncées.
