# Compteur kilométrique truqué : que faire ?

## En bref
Un compteur kilométrique truqué est une fraude courante dans la vente de voitures d’occasion. Vous avez des droits : le vendeur doit fournir un kilométrage exact et précis. Si vous suspectez une fraude, agissez rapidement pour faire valoir vos droits.

## De quoi s'agit-il ?
Un compteur kilométrique truqué signifie que le kilométrage affiché sur le véhicule ne correspond pas au kilométrage réel. Cette pratique est illégale et constitue une tromperie envers l’acheteur. Elle peut concerner les ventes chez les professionnels comme les particuliers.

## Quels sont vos droits ?
La loi impose au vendeur de fournir un kilométrage exact et précis. Le certificat de cession doit mentionner le kilométrage. Si le compteur est truqué, vous pouvez demander l’annulation de la vente ou une réduction du prix. Le vendeur peut être sanctionné pénalement pour tromperie.

## Que faire concrètement ?
1. **Vérifiez le kilométrage** : Comparez le kilométrage affiché avec les documents du véhicule (carnet d’entretien, contrôle technique).
2. **Consultez HistoVec** : Utilisez le site [HistoVec](https://histovec.interieur.gouv.fr/) pour vérifier l’historique du véhicule, y compris le kilométrage.
3. **Contactez le vendeur** : Demandez des explications écrites et exigez une correction du kilométrage ou une réduction du prix.
4. **Signalez la fraude** : Si le vendeur ne coopère pas, signalez la fraude via [SignalConso](https://signal.conso.gouv.fr/fr/voiture-vehicule/faire-un-signalement) et déposez une plainte sur [THÉSEE](https://www.thesee.gouv.fr).

## Exemples / cas concrets
**Situation** : "J'ai acheté une voiture d'occasion chez un professionnel et le compteur indique 80 000 km, mais le carnet d'entretien montre des visites pour 120 000 km."
**Ce que vous pouvez faire** : Demandez au vendeur de justifier la différence. Si aucune explication satisfaisante n'est fournie, signalez la fraude via [SignalConso](https://signal.conso.gouv.fr/fr/voiture-vehicule/faire-un-signalement) et déposez une plainte.
**Résultat attendu** : Le vendeur doit corriger le kilométrage ou vous indemniser. Si la fraude est avérée, le vendeur peut être sanctionné pénalement.

**Situation** : "J'ai acheté une voiture d'occasion à un particulier et le compteur indique 60 000 km, mais le contrôle technique montre des usures correspondant à un kilométrage bien plus élevé."
**Ce que vous pouvez faire** : Contactez le vendeur pour exiger une explication. Si le vendeur refuse de coopérer, signalez la fraude via [SignalConso](https://signal.conso.gouv.fr/fr/voiture-vehicule/faire-un-signalement) et déposez une plainte.
**Résultat attendu** : Le vendeur doit justifier le kilométrage ou vous indemniser. Si la fraude est avérée, le vendeur peut être sanctionné pénalement.
