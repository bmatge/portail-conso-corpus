# Que faire si votre fournisseur d'énergie refuse votre chèque énergie ?

## En bref
Le chèque énergie est une aide nominative pour payer vos factures d'énergie. Si votre fournisseur refuse de l'accepter, vous pouvez contester cette décision. Voici les étapes à suivre pour faire valoir vos droits.

## De quoi s'agit-il ?
Le chèque énergie est une aide financière destinée à aider les ménages modestes à payer leurs factures d'énergie (électricité, gaz, fioul, bois, etc.). Tous les fournisseurs d'énergie sont tenus de l'accepter comme moyen de paiement. Si votre fournisseur refuse votre chèque énergie, vous pouvez prendre plusieurs mesures pour faire valoir vos droits.

## Quels sont vos droits ?
Le chèque énergie est un moyen de paiement légal pour régler vos factures d'énergie. Les fournisseurs d'énergie sont obligés de l'accepter sans facturer de frais supplémentaires. Si votre fournisseur refuse votre chèque énergie, il enfreint la loi. Vous avez le droit de contester cette décision et de demander le remboursement de tout frais injustement facturé.

## Que faire concrètement ?
1. **Contacter votre fournisseur d'énergie** : Envoyez un courrier recommandé avec accusé de réception à votre fournisseur pour lui rappeler que le chèque énergie est un moyen de paiement légal et qu'il est tenu de l'accepter.
2. **Utiliser le chèque énergie en ligne** : Si possible, utilisez le chèque énergie en ligne via le site [chequeenergie.gouv.fr](https://chequeenergie.gouv.fr). Cela permet de pré-affecter votre chèque à votre facture d'énergie.
3. **Contacter le médiateur national de l'énergie** : Si votre fournisseur continue de refuser votre chèque énergie, contactez le médiateur national de l'énergie. Le médiateur peut intervenir pour résoudre le litige à l'amiable.
4. **Saisir la DGCCRF** : Si aucune solution n'est trouvée, vous pouvez saisir la DGCCRF pour signaler le refus de votre fournisseur d'accepter le chèque énergie.

## Exemples / cas concrets
**Situation** : "J'ai acheté du fioul et le fournisseur refuse d'accepter mon chèque énergie."
**Ce que vous pouvez faire** : Envoyez un courrier recommandé avec accusé de réception à votre fournisseur pour lui rappeler ses obligations légales. Si le fournisseur persiste, contactez le médiateur national de l'énergie.
**Résultat attendu** : Votre fournisseur accepte le chèque énergie et régularise la situation.

**Situation** : "Mon fournisseur d'électricité a facturé des frais pour le rejet de mon chèque énergie."
**Ce que vous pouvez faire** : Demandez le remboursement des frais injustement facturés en envoyant un courrier recommandé avec accusé de réception. Si le fournisseur ne répond pas, contactez le médiateur national de l'énergie.
**Résultat attendu** : Votre fournisseur rembourse les frais et accepte le chèque énergie pour les paiements futurs.

**Situation** : "Je n'ai pas reçu mon chèque énergie malgré mon éligibilité."
**Ce que vous pouvez faire** : Vérifiez votre éligibilité sur le site [chequeenergie.gouv.fr](https://chequeenergie.gouv.fr). Si vous êtes éligible, contactez le service client du chèque énergie pour signaler le problème.
**Résultat attendu** : Vous recevez votre chèque énergie par courrier ou en ligne.
