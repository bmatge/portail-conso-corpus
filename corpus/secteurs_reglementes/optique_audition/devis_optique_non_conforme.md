# Le devis en optique ou audition n'est pas conforme au modèle réglementaire 100% Santé : que faire ?

## En bref
Votre devis pour des lunettes, lentilles ou aides auditives doit respecter un modèle réglementaire. Si ce n'est pas le cas, vous pouvez exiger un devis conforme. Le professionnel doit inclure une offre 100% Santé (reste à charge zéro) et une autre offre. Vous avez 2 mois pour accepter le devis.

## De quoi s'agit-il ?
Le devis normalisé est obligatoire pour toute vente d'aides auditives ou de produits d'optique correctrice (lunettes, lentilles). Ce modèle standardisé permet de comparer clairement les offres, notamment l'offre 100% Santé (reste à charge zéro) et une autre offre.

**Qui est concerné ?**
- Tous les professionnels de santé (opticiens, audioprothésistes)
- Tous les consommateurs achetant des produits d'optique ou d'aides auditives

**Périmètre couvert**
- Lunettes correctrices
- Lentilles correctrices
- Aides auditives

**Périmètre non couvert**
- Produits non couverts par le dispositif 100% Santé
- Prestations non liées à l'optique ou à l'audition

## Quels sont vos droits ?
Le professionnel doit vous remettre un devis conforme au modèle réglementaire. Ce devis doit obligatoirement comprendre :
- Une offre 100% Santé (reste à charge zéro)
- Une autre offre (tarifs libres)
- Toutes les informations essentielles (marque, modèle, prix, remboursements, etc.)

Le devis doit être valable au minimum 2 mois. Le professionnel ne peut pas modifier le modèle réglementaire (supprimer ou ajouter des mentions).

## Que faire concrètement ?
1. **Vérifiez le devis** : assurez-vous qu'il contient bien les deux offres (100% Santé et autre offre) et toutes les informations obligatoires.
2. **Demandez un devis conforme** : si le devis n'est pas conforme, exigez un devis respectant le modèle réglementaire.
3. **Comparez les offres** : utilisez le devis pour comparer les offres et faire votre choix.
4. **Signalez le problème** : si le professionnel refuse de vous remettre un devis conforme, signalez-le via SignalConso ou contactez un médiateur sectoriel.

## Exemples / cas concrets

**Situation** : "J'ai acheté des lunettes et le devis ne mentionne pas l'offre 100% Santé."
**Ce que vous pouvez faire** : Demandez un nouveau devis conforme au modèle réglementaire. Le professionnel est obligé de vous le fournir.
**Résultat attendu** : Vous recevez un devis conforme avec l'offre 100% Santé et une autre offre.

**Situation** : "Le devis pour mes aides auditives ne respecte pas le modèle standardisé."
**Ce que vous pouvez faire** : Signalez le problème via [SignalConso](https://signal.conso.gouv.fr). Le professionnel doit respecter le modèle réglementaire.
**Résultat attendu** : Le professionnel corrige le devis ou est sanctionné pour non-respect de la réglementation.

**Situation** : "Le devis ne mentionne pas les montants des remboursements et le reste à charge."
**Ce que vous pouvez faire** : Exigez un devis complet avec toutes les informations obligatoires. Le professionnel doit fournir ces informations.
**Résultat attendu** : Vous recevez un devis complet avec tous les détails nécessaires pour faire votre choix.
