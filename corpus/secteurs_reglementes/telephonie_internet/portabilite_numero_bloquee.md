# Portabilité du numéro bloquée par l’ancien opérateur — que faire ?

## En bref
Vous avez demandé la portabilité de votre numéro et l’ancien opérateur ne la réalise pas.  
La loi impose un délai de 1 à 3 jours ouvrés pour le transfert.  
Vous pouvez agir rapidement : mise en demeure, recours au médiateur, puis à l’ARCEP.

## De quoi s’agit‑il ?
La portabilité du numéro vous permet de changer d’opérateur tout en gardant le même numéro de téléphone.  
Elle concerne les lignes fixes et mobiles sous contrat d’abonnement ou de prépaiement.  
Le processus démarre dès que vous avez signé le nouveau contrat et fourni le code de portabilité.  

**Ce qui est couvert** : tout consommateur souhaitant conserver son numéro, que le changement se fasse en boutique, en ligne ou par téléphone.  
**Ce qui n’est pas couvert** : les numéros temporaires, les lignes d’urgence (112, 15…) et les numéros attribués à des services collectifs (ex. numéros de salle).  

Des blocages peuvent se produire pour trois raisons principales :  
1. L’ancien opérateur retarde volontairement le transfert.  
2. Le code de portabilité n’est pas communiqué dans les 24 h.  
3. Le numéro est perdu ou dupliqué pendant le passage d’un réseau à l’autre.  

Dans tous les cas, le consommateur reste maître de son choix et dispose de recours précis.

## Quels sont vos droits ?
Le texte législatif impose à l’ancien opérateur une **obligation de résultat** : le numéro doit être transféré dans les 1 à 3 jours ouvrés.  
Il doit vous remettre le code de portabilité sous 24 heures après votre demande.  
Si le code n’est pas fourni, le transfert ne peut pas être lancé et le retard constitue une violation du droit du consommateur.  

Vous avez le droit d’exiger :  

* Le respect du délai légal.  
* Le versement éventuel de dommages‑intérêts si le retard vous cause un préjudice (perte d’appels professionnels, frais supplémentaires, etc.).  
* La facturation unique : vous ne devez pas être facturé par les deux opérateurs pendant la période de transition.  

**Exceptions** : en cas de force majeure reconnue (catastrophe naturelle, panne généralisée du réseau) l’opérateur peut justifier un délai supplémentaire. La force majeure doit être prouvée et communiquée rapidement.  

En l’absence de justification, le consommateur peut saisir les autorités compétentes sans frais supplémentaires.

## Que faire concrètement ?
1. **Envoyer une mise en demeure** à votre ancien opérateur.  
   Rédigez un courrier recommandé avec accusé de réception.  
   Mentionnez votre demande de code de portabilité, le délai légal de 24 h et le risque de recours.  

2. **Contacter votre nouveau opérateur** dès réception du code.  
   Transmettez‑lui le code de portabilité et demandez le lancement du transfert.  
   Vérifiez que le nouveau service indique la date prévue de mise en service.  

3. **Faire intervenir le médiateur des communications électroniques** si l’ancien opérateur ne répond pas sous 24 h ou refuse de fournir le code.  
   Le médiateur examine le dossier, propose une solution amiable et peut contraindre l’opérateur à respecter ses obligations.  

4. **Saisir l’ARCEP** en dernier recours.  
   Déposez votre réclamation via le formulaire en ligne ou par courrier.  
   L’ARCEP peut sanctionner l’opérateur, ordonner le respect du délai et, le cas échéant, infliger une amende.  

Conservez toutes les preuves : copies de courriers, captures d’écran de vos échanges, factures montrant la double facturation éventuelle.

## Exemples / cas concrets

**Situation** : « J’ai demandé le code de portabilité, mais l’ancien opérateur ne répond pas. »  
**Ce que vous pouvez faire** : envoyez un courrier recommandé en rappelant le délai de 24 h, puis, si aucune réponse, contactez le médiateur des communications électroniques.  
**Résultat attendu** : l’opérateur fournit le code sous 48 h et le nouveau fournisseur effectue le transfert dans les 3 jours ouvrés.

**Situation** : « Mon numéro a disparu pendant la portabilité, je ne peux plus appeler ni recevoir d’appels. »  
**Ce que vous pouvez faire** : prévenez immédiatement le nouveau opérateur, demandez le rétablissement urgent du numéro et, en parallèle, informez le médiateur.  
**Résultat attendu** : le numéro est réactivé rapidement, les facturations sont suspendues et le médiateur veille à ce que le préjudice soit indemnisé.

**Situation** : « Je suis facturé par les deux opérateurs pendant la période de portabilité. »  
**Ce que vous pouvez faire** : réclamez la résiliation immédiate du contrat chez l’ancien opérateur, exigez le remboursement des frais indus et, si l’opérateur refuse, saisissez le médiateur puis l’ARCEP.  
**Résultat attendu** : l’ancien opérateur arrête la facturation, vous êtes remboursé des sommes indûment perçues et le nouveau opérateur assure la continuité du service.

**Situation** : « L’ancien opérateur refuse de me remettre le code de portabilité, prétextant une erreur technique. »  
**Ce que vous pouvez faire** : mentionnez dans votre mise en demeure que l’erreur technique ne justifie pas le non‑respect du délai légal, puis saisissez le médiateur.  
**Résultat attendu** : le médiateur contraint l’opérateur à délivrer le code, le transfert s’effectue et vous obtenez éventuellement une compensation pour le désagrément.

## Où signaler / à qui s’adresser ?
- **Priorité 1 – ARCEP** : [Saisir l’ARCEP](https://www.arcep.fr/actualites/actualites-et-communiques/detail/n/espace-utilisateurs.html)  
  Vous pouvez déposer votre réclamation en ligne ou par courrier recommandé.  

- **Priorité 2 – Médiation de la consommation** : [Saisir le médiateur de la consommation](https://www.economie.gouv.fr/mediation-conso)  
  Ce service intervient lorsque le litige porte sur la facturation ou la résiliation.  

- **Médiateur sectoriel – Communications électroniques** : [Saisir le médiateur des communications électroniques](https://www.mediateur-telecom.fr)  
  Il traite spécifiquement les problèmes de portabilité, de code de portabilité et de double facturation.  

- **Votre nouveau opérateur** : contactez le service client dès que vous avez le code.  
  Demandez une confirmation écrite du planning de transfert.  

> **Question pivot** : si vous avez souscrit votre nouveau contrat en ligne, privilégiez le contact via le formulaire dédié du nouveau fournisseur.  
> Si vous avez changé d’opérateur en boutique, passez directement par le service client du point de vente.
