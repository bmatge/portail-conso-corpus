# Portabilité du numéro bloquée par l'ancien opérateur : que faire ?

## En bref
La portabilité du numéro permet de conserver son numéro de téléphone lors d'un changement d'opérateur. Si votre ancien opérateur bloque cette portabilité, vous avez des droits. Agissez rapidement pour faire valoir vos droits et résoudre le problème.

## De quoi s'agit-il ?
La portabilité du numéro de téléphone est un droit qui permet de changer d'opérateur tout en conservant son numéro. Cependant, il arrive que l'ancien opérateur bloque cette portabilité, ce qui peut entraîner des retards, des pertes de service ou des facturations simultanées. Ce problème concerne tous les consommateurs souhaitant changer d'opérateur tout en conservant leur numéro de téléphone.

## Quels sont vos droits ?
Selon la loi, la portabilité du numéro doit être effectuée sous 1 à 3 jours ouvrés. L'ancien opérateur doit fournir toutes les informations nécessaires pour que la portabilité se déroule sans encombre. Si l'ancien opérateur bloque intentionnellement ou retarde la portabilité, il enfreint la réglementation. Vous avez le droit d'exiger que la portabilité soit effectuée dans les délais légaux et de demander des dommages et intérêts en cas de préjudice.

## Que faire concrètement ?
1. **Contactez votre ancien opérateur** : Informez-le par écrit (courrier recommandé avec accusé de réception) de votre intention de porter votre numéro et demandez confirmation de la réception de votre demande.
2. **Demandez un code de portabilité** : Votre ancien opérateur doit vous fournir un code de portabilité sous 24 heures. Ce code est nécessaire pour que le nouveau opérateur puisse transférer votre numéro.
3. **Contactez votre nouveau opérateur** : Fournissez-lui le code de portabilité et demandez-lui de procéder au transfert. Le nouveau opérateur doit effectuer la portabilité sous 1 à 3 jours ouvrés.
4. **Saisissez l'ARCEP ou le médiateur** : Si l'ancien opérateur refuse de fournir le code de portabilité ou retarde intentionnellement le processus, saisissez l'[ARCEP](https://www.arcep.fr/actualites/actualites-et-communiques/detail/n/espace-utilisateurs.html) ou le [médiateur des communications électroniques](https://www.mediateur-telecom.fr).

## Exemples / cas concrets

**Situation** : "J'ai demandé à mon ancien opérateur de me fournir le code de portabilité, mais il refuse de me le donner."
**Ce que vous pouvez faire** : Envoyez un courrier recommandé avec accusé de réception à votre ancien opérateur pour exiger le code de portabilité. Si l'opérateur ne répond pas sous 24 heures, contactez l'[ARCEP](https://www.arcep.fr/actualites/actualites-et-communiques/detail/n/espace-utilisateurs.html) pour signaler le problème.
**Résultat attendu** : Votre ancien opérateur doit vous fournir le code de portabilité sous 24 heures. Si ce n'est pas le cas, l'ARCEP peut intervenir pour faire respecter la réglementation.

**Situation** : "Mon numéro a été perdu pendant la portabilité."
**Ce que vous pouvez faire** : Contactez immédiatement votre nouveau opérateur pour signaler le problème. Demandez-lui de rétablir votre numéro dans les plus brefs délais. Si le problème persiste, contactez le [médiateur des communications électroniques](https://www.mediateur-telecom.fr).
**Résultat attendu** : Votre nouveau opérateur doit rétablir votre numéro dans les plus brefs délais. Si ce n'est pas le cas, le médiateur peut intervenir pour résoudre le problème.

**Situation** : "Je suis facturé par les deux opérateurs pendant la portabilité."
**Ce que vous pouvez faire** : Contactez votre ancien opérateur pour demander la résiliation de votre ancien contrat. Si l'ancien opérateur refuse, contactez le [médiateur des communications électroniques](https://www.mediateur-telecom.fr).
**Résultat attendu** : Votre ancien opérateur doit résilier votre ancien contrat et cesser de vous facturer. Si ce n'est pas le cas, le médiateur peut intervenir pour résoudre le problème.
