# Alimentation, cosmétiques, produits d'entretien : comment éviter les substances indésirables ? avec l'UFC-Que choisir

**- Alimentation**

12/07/2024
**- Alimentation**

Bonjour, comment reconnaître les produits bons pour ma santé quand je fais mes courses ?

Pour avoir une alimentation saine, acheter des cosmétiques ou des produits d’entretien sans nocivité pour sa santé avec un faible impact sur l’environnement, voici comment éviter les substances indésirables.


Au-delà des messages publicitaires et du marketing sur les emballages des aliments, lisez les étiquettes pour bien comparer les produits.


Pour les courses alimentaires, privilégiez les aliments bruts. Mais si vous achetez des aliments industriels, préférez les produits les moins transformés et vérifiez le Nutri-Score qui traduit l’intérêt nutritionnel, grâce à un code couleur immédiatement compréhensible : vert pour les aliments à favoriser et orange ceux à limiter. Mais certains fabricants n’affichent pas le Nutri-score.


Quant aux ingrédients entrant dans la composition, notamment les additifs, à moins d’être un expert, il reste souvent difficile de connaître leur dangerosité. Et c’est encore plus vrai pour les cosmétiques et les détergents dont les compositions sont particulièrement difficiles à comprendre.


Pour éviter les substances toxiques, vous pouvez aussi vous munir de votre smartphone.


Vous pouvez télécharger gratuitement notre application "[Quelproduit](https://www.quechoisir.org/application-mobile-quelproduit-n84731/)" qui vous aidera à choisir vos aliments, cosmétiques et détergents. En scannant le code-barre d’un produit, vous trouverez son évaluation, faite par nos experts.


Les critères ? Le niveau de risque des substances indésirables comme :


- l’octocrylène, un filtre UV présent dans certaines crèmes solaires,
- ou le butylphényl methylpropronial, un parfum présent dans les produits d’entretien suspecté d’être perturbateur endocrinien.


Cette application note les cosmétiques en fonction de leur mode d’utilisation et de la sensibilité des profils de consommateur.


Pour les produits alimentaires, elle calcule systématiquement le Nutri-Score. Les produits reçoivent également une note environnementale qui tient compte des impacts des substances présentes sur la faune et la flore. Si le produit est mal noté, pas de panique, l’application vous propose des produits comparables sans risques pour la santé.
