# Ce sont vos données - Prenez le contrôle - Guide du citoyen sur la protection des données dans l’Union Européenne

# Ce sont vos données - Prenez le contrôle - Guide du citoyen sur la protection des données dans l’Union Européenne

![](https://www.inc-conso.fr/sites/default/files/ce-sont-vos-donnees.png)


## Objectif(s)

Sensibiliser et informer les citoyens sur la protection des données personnelles au sein de l’Union européenne.

## Contenu

Ce guide de 16 pages (21 cm x15 cm), édité en 2018, présente le règlement général sur la protection des données (RGPD). Il informe sur les différents droits des citoyens à savoir :


- Protéger vos données personnelles.
- Le droit de savoir qui traite quoi et pourquoi.
- Le droit d’accéder à vos données.
- Le droit de vous opposer.
- Le droit de corriger vos données.
- Le droit de faire supprimer vos données et le droit à l’oubli.
- Le droit d’avoir votre mot à dire en cas de décisions automatisées.
- Le droit de transférer vos données.

Un encadré "Comment prendre contact avec l’Union européenne ?", complète le guide.

## Observations du Comité Paritaire

Document d’information qui reprend les informations concernant le "RGPD" en plus synthétique.

Il est clair, avec des exemples bien choisis pour faciliter la compréhension du sujet.
