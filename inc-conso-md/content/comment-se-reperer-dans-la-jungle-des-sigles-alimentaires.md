# Comment se repérer dans la jungle des sigles alimentaires ?

**- Alimentation**

Les labels fleurissent sur les emballages des produits mais quels sont-ils ? Que signifient-ils ?


L’association Léo Lagrange de Défense des Consommateurs fait le point sur les signes de qualité dans le guide "Se repérer dans la jungle des sigles alimentaires", afin d’aider les consommateurs à mieux comprendre et identifier les caractéristiques de chacun d’eux.


![](https://www.inc-conso.fr/sites/default/files/homme-courses-label-800.png)



Ce guide décrypte les labels critères par critères. Il vous aiguille dans vos choix pour une alimentation de qualité, saine, éthique, respectueuse des besoins nutritionnels et de l’environnement.


![](https://www.inc-conso.fr/sites/default/files/brochure-sigles-alldc-couv-guide-250.png)


![](https://www.inc-conso.fr/sites/default/files/brochure-sigles-alldc-couv-guide-250.png)

1 - Les labels écologiques

2 - Les mentions valorisantes

3 - Les labels nutritionnels

4 - Les labels éthiques

5 - Les labels d’origine géographique

6 - Les labels qualité supérieure

7 - Les labels marketing
