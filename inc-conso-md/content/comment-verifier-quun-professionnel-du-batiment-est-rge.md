# Comment vérifier qu'un professionnel du bâtiment est "RGE" ?

**- Énergie/environnement**

Pour bénéficier de certaines aides, primes ou prêts, vous devez confier les travaux de rénovation énergétique à un professionnel du bâtiment qualifié "RGE".


Malheureusement, certains "margoulins" se disent "RGE" sans l'être réellement. Dans ce cas, vous perdez le bénéfice d'aides auxquelles vous aviez droit.


![](https://www.inc-conso.fr/sites/default/files/expert-rge-800.png)



C'est pourquoi il est nécessaire de vérifier par vous-même si le professionnel est bien qualifié "RGE". Pour cela, vous pouvez aller [sur le site "France Renov"](https://france-renov.gouv.fr/fr/trouvez-un-professionnel).



En cliquant sur l'onglet "Rechercher un professionnel", vous pouvez retrouver une entreprise ses coordonnées.


![](https://www.inc-conso.fr/sites/default/files/rge-site-700.png)



Vous pourrez alors trouver un professionnel, mais également vérifier le statut de la société : existe-t-elle toujours ? Est-elle toujours RGE ? Quel est son domaine de travaux ?


Attention : Il est essentiel de vérifier la date de valdité de la mention "RGE", ainsi que les travaux couverts ! Pour cela, téléchargez l'attestation disponible sur le site. Elle mentionne le domaine de travaux, pour lequel la société est "RGE", la période de validité de la mention, ainsi que des éléments très précieux, comme les coordonnées de l'entreprise ou le nom de son assureur.
