# FiligraneFacile : un outil simple pour protéger vos documents

**- Droit/justice**



04/08/2025
**- Droit/justice**



L’utilisation frauduleuse de documents et l’usurpation d’identité sont en nette augmentation. Pour lutter contre ces dérives, le site **FiligraneFacile** a été lancé en septembre 2022. Il permet d’ajouter un filigrane numérique sur le document de votre choix afin de le protéger. Xavier Serre, de l’association Familles Rurales 58 nous explique comment l’utiliser.

**FiligraneFacile** est un site officiel du gouvernement qui permet d’insérer une mention transparente personnalisée sur des documents sensibles. Ces documents deviennent alors inutilisables pour tout autre usage que celui précisé par le filigrane.


Par exemple, s’il est indiqué "destiné à la location immobilière exclusivement", il ne pourra pas être utilisé pour ouvrir un compte bancaire. Ce système réduit significativement les risques d’usurpation. Mais restez tout de même vigilant lors de l’envoi de vos documents, surtout si vous ne connaissez pas le destinataire !


Sachez que **FiligraneFacile** a été conçu avec un haut niveau de sécurité. Le document que vous mettez en ligne, ainsi que sa version filigranée, est automatiquement supprimé des serveurs après le premier téléchargement, ou au bout de vingt-quatre heures s’il n’y en a pas eu. Cela limite les risques en cas de cyberattaque.


Pour utiliser cet outil :


- rendez-vous sur le site
[filigrane.beta.gouv.fr,](http://filigrane.beta.gouv.fr) - sélectionnez le fichier de votre choix sur votre appareil,
- renseignez le texte du filigrane dans la zone prévue,
- cliquez sur "Ajouter le filigrane",
- cliquez "Télécharger le document filigrané".


Vous obtiendrez un fichier PDF au format portrait.


Le site **FiligraneFacile** est encore en phase de développement. Si vous avez des suggestions ou des remarques pour l’améliorer, vous pouvez les envoyer à l’adresse suivante : [contact@filigrane.beta.gouv.fr](http://contact@filigrane.beta.gouv.fr). Pour sécuriser vos documents, adoptez le réflexe filigrane !
