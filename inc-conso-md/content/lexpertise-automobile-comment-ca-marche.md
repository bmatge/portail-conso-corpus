# L'expertise automobile, comment ça marche ?

# L'expertise automobile, comment ça marche ?

**- Automobile/2 roues**

02/12/2019
**- Automobile/2 roues**

Un accident de voiture est vite arrivé. Alez sachez de quelle façon fonctionne l’expertise automobile.

Le rôle de l’expert automobile est :


- de vérifier, de façon indépendante, la vraisemblance d’un l’accident par rapport à de ce que vous avez déclaré,
- de décrire les dégâts,
- de déterminer qui est responsable des dommages,
- il doit aussi convenir, en accord avec le garagiste, de la méthode et du coût des réparations.

L’expert intervient après déclaration du sinistre pour chiffrer les dommages afin que l’assureur puisse vous rembourser.


Attention cependant si vous êtes entièrement responsable de l’accident et que vous ne bénéficiez pas de garanties facultatives. Dans ces cas là, vous n’obtiendrez rien. Nul besoin d’expertise donc. L’expert Intervient seulement si vous êtes entièrement responsable de l’accident et que vous avez une assurance facultative ou si vous êtes partiellement responsable ou non responsable.


L’expert est alors désigné par votre assureur et le coût de cette première expertise est réglé par la compagnie d’assurance. L’expert doit ensuite vous adresser une copie du rapport d’expertise.


Si vous n’êtes pas d’accord sur l’imputation des dommages, sur le prix des réparations ou la valeur du véhicule, vous pouvez faire une contre-expertise avec l’expert de votre choix. Vous devrez alors régler ses honoraires. Si cela est prévu dans le contrat, cette contre-expertise peut toutefois être gratuite pour l’assuré.


Enfin, si les deux experts ne parviennent pas à se mettre d’accord, les contrats prévoient généralement la désignation d’un troisième expert. Ce dernier a pour mission de trancher le litige.



**En résumé **:


- l’expert n’intervient que dans certains cas,
- le coût est réglé par votre assureur,
- en cas de désaccord, vous devez effectuer une contre expertise à vos frais.
