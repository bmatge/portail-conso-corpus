# Additifs : savoir les repérer sur les étiquettes avec Foodwatch

**- Alimentation**

08/04/2024
**- Alimentation**

Bonjour, j’aimerais en savoir plus sur les additifs alimentaires et la manière de les repérer sur les étiquettes ?

Merci pour votre question qui met en avant un point important : la mention obligatoire des additifs sur les emballages des denrées alimentaires.

Conservateur, antioxydant, exhausteur de goût, agent de texture… Voici quelques-unes des propriétés pour lesquelles les additifs alimentaires sont utilisés dans les produits transformés.

**En Europe, on en recense environ 320 autorisés. Mais comment ces derniers sont-ils mentionnés sur les étiquettes ?**

Et bien de plusieurs manières ! Pour commencer, leur mention est obligatoire, mais impopulaires chez les industriels qui recourent souvent à des astuces pour essayer de masquer leur présence.

Ils peuvent la mentionner dans la liste des ingrédients :

- soit par leur code : un E suivi de 3 ou 4 chiffres,
- soit par l’un de leur nom. C’est le cas, par exemple du Nitrate de potassium dont le code est E252 et qui peut aussi avoir comme dénomination "salpêtre", "nitrate depotasse", "nitre", ou encore "azoate de potasse.

**S’il est difficile de connaitre tous les additifs par leur numéro de code ou leur nom scientifique, alors les produits qui affichent un label sont-ils de bons repères pour les consommateurs ?**

Pas nécessairement ! Ce n’est pas parce que sur l’emballage vous voyez la mention d’un label ou d’une indication. Par exemple, une boite de gâteaux sans colorant ne veut pas dire qu’il n’y a pas d’autres additifs dedans.

La seule manière de le savoir, c’est de retourner votre emballage et de regarder la liste des ingrédients et d’essayer de les repérer.

**Les additifs présents dans les denrées alimentaires sont pourtant autorisés. Alors pourquoi mieux vaut les éviter ?**

Vous avez raison, [l’Autorité européenne de sécurité des aliments](https://european-union.europa.eu/institutions-law-budget/institutions-and-bodies/search-all-eu-institutions-and-bodies/european-food-safety-authority-efsa_fr) évalue si un additif peut être utilisé ou non au sein de l’Union européenne. Cela n’empêche malheureusement pas l’autorisation d’additifs controversés .C’est le cas par exemple des carraghénanes ou des additifs à base de nitrites et de nitrates.

**Alors quel conseil peut-on donner pour les éviter ou en tout cas, réduire leur consommation ?**

Et bien, regardez la liste des ingrédients et choisissez si vous le pouvez de préférence les produits dont la liste est la plus courte, ce qui généralement signifie qu’ils sont moins transformés.
