# Comparez les syndics !

**- Logement/immobilier**

Le conseil syndical va lancer un appel d'offres auprès de plusieurs syndics. Ces derniers vont lui transmettre un contrat récapitulant leurs honoraires. Il est important d'étudier ces documents pour déterminer le coût, mais également les missions qui sont proposées par ces professionnels.

L'INC a établi un tableau récapitulant les différentes prestations incluses dans la mission du syndic. L'objectif est de comparer les différentes offres et de choisir le professionnel, qui correspond le plus aux besoins initiaux de la copropriété.** Vous pouvez comparer jusqu'à 9 syndics.**

Il est possible de personnaliser le simulateur en remplissant les lignes vides.

Le simulateur est au format xlsx et le logiciel Excel est requis pour ouvrir le calculateur.

Si vous ne l'avez pas, des équivalents gratuits existent.

Une fois le simulateur téléchargé, enregistrez-le sur votre ordinateur.
