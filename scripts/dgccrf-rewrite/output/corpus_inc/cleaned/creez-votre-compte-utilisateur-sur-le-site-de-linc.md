# Créez votre compte utilisateur sur le site de l'INC

Créer son compte sur le site de l'INC vous permet :

- de vous inscrire
pour être tenu informé de toutes les nouveautés du site (fiches pratiques, lettres types, vidéos CONSO MAG, articles d'actualité...) sans attendre les lettres d'informations hebdomadairs.**aux alertes**

- de vous inscrire
"[aux newsletters](/content/newsletters)**Lettre générale**" (toutes les informations du site) et/ou "**Lettre spéciale Education**" (les nouveaux outils et activités pédagogiques), - de retrouver
en cliquant sur l’étoile présente sur chaque page du site pour les retrouver à la prochaine connexion,[vos articles préférés](/bookmarks)

- d'
(fiche pratique, lettre type, question-réponse, simulateur, articles...),**être averti de la mise à jour d'un article**

> Retrouvez à tous moments [vos alertes ](/bookmarks)pour y accéder rapidement.
