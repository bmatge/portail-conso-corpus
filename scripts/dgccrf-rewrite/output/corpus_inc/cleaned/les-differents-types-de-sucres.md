# Les différents types de sucres

**- Alimentation**

Le **sucre** est une substance utilisée pour donner du goût aux aliments : le goût sucré tant apprécié. Il est fabriqué en France métropolitaine à partir de betterave ou de canne à sucre dans les Antilles.

Le sucre existe aussi sous différentes formes souvent méconnues du public, utilisées dans l’industrie agroalimentaire :

**saccharose,****sirop de glucose,****fructose, dextrose**…

Ces substances représentent toutes la même chose : des sucres que nous avons décidé de nommer "sucre cachés".

De plus, des additifs particuliers peuvent être utilisés pour augmenter le goût sucré d’un produit. Les industries agroalimentaires utilisent des **édulcorants** qui ont la particularité d’avoir des pouvoirs sucrants très élevés. On peut citer :

**Les édulcorants basses calories**comme l’Aspartame (E951) qui permet de sucrer un produit en utilisant de faibles quantités.**Les édulcorants avec calories**qui ont l’avantage de ne pas faire augmenter la glycémie (utilisés, entre autres, pour les diabétiques).
