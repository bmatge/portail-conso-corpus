# Mieux comprendre les objectifs de développement durable : quiz

![](/sites/all/themes/custom/inc/assets/images/icons/pedago_3_coeur.jpg)

## Objectif(s)

Sensibiliser sur le développement durable.

## Contenu

Ce quiz est composé de plusieurs questions sur différents thèmes du développemnt durable (climat, santé, éducation,biodiversité, etc.).

## Observations du Comité Paritaire

Bon quiz de 170 questions sur le climat. Cependant, il demande un temps de travail à l’animateur qui devra choisir les questions selon la cible définie pour son apprentissage.

Les différentes questions (choix multiples) sont bien élaborées.

Les réponses, suivies de commentaires, sont de compréhension facile, permettent une bonne culture générale et suscitent l’envie d’agir sur le sujet.

Sur les 170 questions que comporte l'outil, il est conseillé de réserver les plus complexes aux initiées.
