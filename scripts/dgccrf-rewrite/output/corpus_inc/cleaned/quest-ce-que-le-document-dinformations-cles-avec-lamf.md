# Qu’est-ce que le document d’informations clés ? avec l'AMF

**- Banque/argent**

24/01/2024
**- Banque/argent**

Mon conseiller financier m’a remis un document d’informations clés. A quoi sert-il ? Réponse avec Olivier Eon de l'Autorité des marchés financiers.

Le **document d'informations clés** (DIC) est un document standardisé au niveau européen. Il vous permet de retrouver les informations essentielles sur un placement collectif comme un fonds, une sicav ou une SCPI.

Le DIC contient les principales caractéristiques du placement comme :

- sa stratégie,
- sa durée de détention recommandée,
- son niveau de risque à travers une échelle qui va de 1 pour les placements les moins risqués, à 7 pour les plus risqués.

Le DIC présente également le détail des frais qui seront appliqués au placement.

Son format étant standardisé, il permet de comparer plusieurs placements avant de se décider.

Le professionnel qui vous recommande un placement financier doit vous fournir son document d'informations clés.

Si vous choisissez vous-même vos placements, sans bénéficier du conseil d'un professionnel, c’est généralement l’intermédiaire financier par lequel vous passez qui le met à votre disposition. Vous pouvez également retrouver le DIC sur le site internet de la société qui gère le placement dans lequel vous envisagez d'investir.

> Pour en savoir plus, contactez [AMF Epargne Info Service](https://www.amf-france.org/fr/espace-epargnants/lamf-et-vous/epargne-info-service).
