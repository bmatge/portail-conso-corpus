# Vous souhaitez demander à votre opérateur une facture détaillée de vos communications

**- Internet/multimédia**

Vous avez souscrit un contrat de téléphonie ou de fourniture d’accès à Internet. A votre demande, votre opérateur peut vous délivrer gratuitement des factures détaillées de vos communications, par courrier ou par courriel, si vous avez accepté ce mode de transmission (article 1126 du code civil). Ces factures doivent comporter un niveau de détail suffisant pour permettre la vérification des montants facturés.

Elles ne mentionnent pas les appels à destination des numéros gratuits et n’indiquent pas les quatre derniers chiffres des numéros appelés, sauf si vous en faites la demande (article D. 98-5 du code des postes et des communications électroniques).

**Lettre recommandée avec avis de réception**

Référence du contrat :

Madame, Monsieur,

Titulaire d’un abonnement (précisez : de téléphonie ou d’accès à Internet), je souhaiterais obtenir les factures détaillées de mes communications.

(Facultatif) Conformément à l’article 1126 du code civil, je donne mon accord pour que ces factures me soient délivrées par courriel.

(Facultatif) Conformément à l’article D. 98-5 du code des postes et des communications électroniques, je souhaite que les quatre derniers chiffres des numéros appelés soient inscrits.

En vous remerciant par avance pour la prise en compte de ma demande, je vous prie d’agréer, Madame, Monsieur, l’expression de mes salutations distinguées.

(Signature)
