## Checklist de complétude (réécriture)

Avant de finaliser la fiche réécrite, vérifie que :

1. [ ] Le titre est une question ou un problème concret du consommateur (pas un titre administratif)
2. [ ] La section "En bref" est autonome (compréhensible sans lire le reste)
3. [ ] Le périmètre est précisé (ce qui EST couvert ET ce qui NE L'EST PAS)
4. [ ] "Que faire concrètement" contient des étapes numérotées avec verbes d'action
5. [ ] Au moins 3 exemples concrets (pour les fiches situation)
6. [ ] TOUTES les URLs de sortie de la taxonomie sont intégrées dans "Où signaler"
7. [ ] Le médiateur sectoriel est mentionné avec son URL (si fourni dans les sorties)
8. [ ] L'URL SignalConso directe est intégrée (si un champ `signalconso` est fourni)
9. [ ] Pas de jargon juridique non expliqué dans le corps de la fiche
10. [ ] Les références légales sont UNIQUEMENT dans "Pour aller plus loin"
11. [ ] Le format est du Markdown pur (pas de HTML, pas de balises)
12. [ ] Les sections sans information suffisante sont marquées `[À COMPLÉTER]`
13. [ ] La section "Sujets proches" contient au moins 2 liens vers des fiches d'autres domaines/sous-domaines
14. [ ] La section "À ne pas confondre" identifie au moins 1 situation distincte avec explication claire de la différence

Si un point ne peut pas être vérifié par manque d'information dans les sources, marque `[À COMPLÉTER]` dans la section concernée.

## Exigence de longueur

VÉRIFIE que la fiche atteint la longueur minimale :
- Situation : **1 000 mots minimum**
- Sous-domaine : **1 200 mots minimum**
- Domaine : **1 000 mots minimum**

Si la fiche est trop courte, développe davantage : ajoute des exemples, précise les délais et montants, détaille les cas particuliers, enrichis les étapes de la procédure.
